Accessibility Statement

Demo webshop notice: This website was created for a university exercise and aims to be accessible.

Conformance target

Sockora aims to conform to WCAG 2.2 level AA as far as reasonably possible for a student demo project.

Measures taken

  • Semantic HTML structure (header, nav, main, footer)
  • Keyboard accessible navigation and controls
  • Visible focus indicators
  • Text alternatives (alt text) for images
  • Form labels and accessible input fields
  • Consistent typography and spacing

Testing

The website was reviewed using:

  • WAVE browser extension
  • axe DevTools browser extension

Known limitations

  • This webshop is a demo with mocked checkout; no real order processing exists.
  • Some content may not fully cover all edge cases of WCAG 2.2 AA.

Feedback and contact

If you encounter accessibility issues, please contact: accessibility@sockora.example
